# Tugas Metoda Komputasi II
Deskripsi
Repositori ini berisi implementasi tugas pada mata kuliah Metoda Komputasi II. Tugas ini berfokus pada penerapan metode numerik dan komputasi untuk menyelesaikan permasalahan ilmiah melalui pemodelan, analisis data, dan visualisasi.

Tujuan \
Memahami konsep dasar metoda komputasi,
Mengimplementasikan algoritma numerik dalam pemrograman,
Menganalisis dan menginterpretasikan hasil komputasi,
Mengembangkan pemikiran sistematis dalam pemecahan masalah ilmiah,

Ruang Lingkup \
Pemodelan numerik \
Analisis data \ 
Visualisasi hasil \

Tools dan Library \
Python 3.x
NumPy
Matplotlib
SciPy (opsional)
Pandas (opsional)
